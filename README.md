# Spotify_Analysis
Analyse de la popularité des chansons Spotify (2010–2019) avec R


## Contexte

Projet d’analyse de données réalisé dans le cadre d’un portfolio personnel.
L’objectif est d’étudier les facteurs influençant la popularité des chansons Spotify
entre 2010 et 2019.
Analyser les facteurs influençant la popularité des chansons Spotify entre 2010 et 2019.

Objectifs du projet:

Décrire les principales caractéristiques des chansons. 
Analyser leur relation avec la popularité. 
Construire un modèle simple de prédiction de la popularité.

## Données
Dataset Spotify (source : DataCamp).
Environ 600 chansons issues des classements annuels.

## Méthodologie
- Analyse descriptive
- Visualisations exploratoires
- Modelisation
- Régression linéaire

## Résultats principaux
- La danseabilité est positivement associée à la popularité
- Le tempo montre un effet faible
- Le modèle explique une part modérée de la popularité (~30 %)
